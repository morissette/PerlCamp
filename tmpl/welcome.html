Welcome to PerlCamp

This site serves as a resource for new and old developers who wish to learn and/or brush up on Perl. A language which is used to power a lot of the Internet; even today in 2016!

This entire project is open source and can be found at https://github.com/morissette/PerlCamp, I intend for others to help me fix any glaring holes and/or downright wrong information. Anyways enough about me, time to get Perl'ing.

Basic Usage

Login/Register: This enables you to review code that you have executed in the past; however is not necessary for you to use this site.

Glossary: Common terms used in programming and Perl.

Code Review: Enabled upon registration and login

How to use this system

The left pane allows you to write and execute Perl code as you go through this site and shows the output of the execution at the bottom; much like many other code tutorial sites.
